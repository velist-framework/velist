# Workflow Outputs

Folder ini berisi hasil kerja dari agents.

## Structure

| Folder | Agent | Content |
|--------|-------|---------|
| `01-product/` | @ProductAgent | PRD, user stories, roadmap |
| `02-engineering/` | @TechLeadAgent | Tech specs, architecture, API contract |
| `03-tasks/` | @TechLeadAgent | Task breakdowns |
| `04-reports/` | @QAAgent | Test reports |

## File Naming

- Gunakan UPPERCASE untuk file utama: `PRD.md`, `TECH_SPEC.md`
- Gunakan kebab-case untuk tambahan: `api-contract.md`
